# 파일내용반영 흐름

1. (재반영 시) 기존 월간요약과 일간요약 데이터를 스냅샷으로 저장
2. 출입기록, 근태사용이력 데이터 생성, 수정, 삭제
3. 일간요약 생성,수정,삭제 (사용된 근태유형정보는 컬럼으로 저장)
   3-1. 일간요약 생성은 출입기록과 근태사용이력을 기반으로
   3-2. 삭제는 소프트삭제, 수정이력데이터는 모두 일반삭제
4. 월간요약 생성,수정,삭제 후 일간요약과 월간요약의 관계형성

# 스냅샷 적용 흐름

1. 기존 월간요약과 일간요약 데이터를 스냅샷으로 저장
2. 일간요약 생성, 수정,삭제
   2-1. 일간요약 생성은 스냅샷에 저장된 데이터를 기반으로
   2-2. 삭제는 소프트삭제, 수정이력데이터는 모두 일반삭제
3. 월간요약 수정,삭제 후 일간요약과 월간요약의 관계형성
